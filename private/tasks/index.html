01 CosyVoice 2 Real-Time TTS Engine Voice Quality

CosyVoice 2 is installed in WSL2 and proven to generate 50s of cross-lingual speech. The missing piece: a persistent inference server (stdin/stdout or Unix socket) so the Windows pipeline can stream translated text in and receive WAV out without launching a new Python process per utterance. Wire it as a proper CosyVoice2TTSEngine in app/engines/tts/, add a voice-only-cosyvoice2 config profile, and benchmark RTF. This is the highest-leverage voice quality win currently achievable without a Spanish recording.

07 Latency CI Benchmark — GitHub Actions + Trend Chart Infrastructure

On every push to main, GitHub Actions runs the pipeline with MockSTTEngine + MockTTSEngine on a fixed test utterance set, measures per-stage latency (capture→STT→translate→TTS), and compares against the historical baseline. A trend chart is published to GitHub Pages so you can see if a new TTS or translation engine regressed latency. Fails CI if any stage exceeds a 15% regression.
